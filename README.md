---
## 📦 Como instalar e iniciar o projeto

### API (Back-end)

```bash
# Clonar o repositório
$ git clone https://github.com/polyanetuag/NLW9-esports.git

# Entrar na pasta server 
$ cd server

# Instalar as dependências
$ npm i

# Iniciar o servidor
$ npm run dev

```

### Web (Front-end)

```bash

# Entrar na pasta web 
$ cd web

# Instalar as dependências
$ npm i

# Iniciar a aplicação
$ npm run dev

```
